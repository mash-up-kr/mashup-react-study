# 2019년 2월 12일 퀴즈

## Q1

#### 다음 코드의 실행 결과는?

```javascript
function setFruitColor(color) {  
  return (target) => {
    target.prototype.color = color
    return target
  }
}

@setFruitColor('red')
class Apple {  
  mycolor() {
    return this.color
  }
}

@setFruitColor('yellow')
class Banana {  
  mycolor() {
    return this.color
  }
}

const apple = new Apple()  
console.log(apple.mycolor()) 

const banana = new Banana()  
console.log(banana.mycolor()) 
```
