# 벌칙금 제도

## 운영 규칙

1. 스터디 당일 퀴즈의 절반 이상을 못 맞추면 벌칙금 2,000원 추가
2. 스터디 불참 시 벌칙금 2,000원 추가. 이 경우에 퀴즈 또한 FAIL한 것으로 간주하여 최종적으로 4,000원 추가
3. 스터디원 전체가 합의한 경우를 제외하고, 어떠한 사유가 되었든 간에 반드시 벌칙금 납부!
4. 벌칙금은 스터디 회식비 등으로 사용 예정
